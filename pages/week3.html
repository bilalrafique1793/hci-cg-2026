Definition

The Keystroke Level Model (KLM) is a predictive model that estimates the time required for an expert user to complete a task by counting low-level actions such as keystrokes and mouse movements.

Purpose of KLM

KLM helps designers compare different interface designs by predicting which one allows faster task execution.

KLM Operators

  • K – Keystroke
  • P – Pointing
  • H – Homing
  • M – Mental preparation
  • R – System response

Example Task

Task: Open a file

  • M: Decide to open file
  • P: Move mouse to file
  • K: Click file
  • R: System opens file

Assumptions

  • User is expert and error-free
  • No learning or problem solving is included
  • System response time is constant

Use in Interface Design

KLM is used to optimize interaction by reducing unnecessary actions and improving efficiency.
