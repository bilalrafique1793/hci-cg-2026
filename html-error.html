HTML Questions and Answers

Q1: Why should a web page contain only one <h1>?

Answer: The <h1> represents the main topic. Multiple <h1> elements reduce structural clarity.

Q2: What problem occurs if images have no alt text?

Answer: Screen readers cannot describe images, making the page inaccessible.

Q3: Why is excessive use of <div> discouraged?

Answer: <div> elements lack meaning; semantic elements communicate purpose.

Q4: Why is vague link text harmful?

Answer: Screen-reader users rely on descriptive links to understand destinations.

Q5: Why must form inputs be labeled?

Answer: Labels improve usability and accessibility for all users.

Q6: What is wrong with using tables for layout?

Answer: Tables should only present structured data, not page layout.

Q7: Why should heading levels not be skipped?

Answer: Skipping levels disrupts the logical reading order.

Q8: Why is list-based navigation recommended?

Answer: Assistive technologies expect navigation menus to be lists.

Q9: Why is the lang attribute important?

Answer: It allows screen readers to pronounce text correctly.

Q10: What happens if HTML tags are not closed?

Answer: The browser may misinterpret the document structure.
